ferion Architecture

Build pipeline and component tree for a fully static, backend-free React portfolio

Frontend Stack

  • React 19 with the StrictMode root
  • Vite 8 for dev server and bundling
  • Tailwind CSS v4, config lives in CSS
  • motion (Motion / Framer Motion) for animation

Component Sources

  • shadcn for Button, Card, Sheet primitives
  • ForgeUI registry, vendored and patched
  • 21st.dev community macOS Dock, rewritten
  • Aceternity UI Cover effect, rebuilt without tsparticles

Deployment Model

  • Fully static output, zero servers
  • No API routes, no database, no auth
  • Deployable to any static host or CDN
  • Theme preference is the only persisted state
